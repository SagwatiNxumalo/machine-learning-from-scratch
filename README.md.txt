# Machine Learning From Scratch (Python)

This project implements fundamental machine learning algorithms from scratch using Python and NumPy.

## Algorithms Implemented
- Linear Regression
- Logistic Regression

The models are implemented manually without using machine learning libraries such as Scikit-learn in order to understand the underlying mathematics and optimisation process.

## Technologies
- Python
- NumPy

## Concepts Demonstrated
- Gradient Descent
- Statistical modelling
- Model prediction
- Data analysis

## Purpose
This project was developed as part of my Computer Science studies to better understand how machine learning algorithms work internally.
